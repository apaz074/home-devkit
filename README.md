# home-devkit

## Quick Install

To set up your development environment, run the following command in your terminal. This will download and execute the installation script.

```bash
sh <(curl -sSL https://raw.githubusercontent.com/apaz074/home-devkit/main/setup.sh)
```

This command will guide you through the setup process, including installing Nix, cloning the configuration, and personalizing your environment.
